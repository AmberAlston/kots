# Kotsadm

Kotsadm is a locally-installable application that manages [Helm Charts](https://helm.sh) and other Kubernetes applications in a workflow that's built for day-2 operations.

## Contributing

For docs on setting up a dev environment to build and run kotsadm locally, read the [CONTRIBUTING](https://github.com/replicatedhq/kotsadm/blob/master/CONTRIBUTING.md) guide.
